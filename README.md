# 🎵 Análisis exploratorio del catálogo de Spotify
Proyecto de análisis de datos. Enfocado en el análisis exploratorio de datos (EDA) y en el *data storytelling*.

El objetivo de este proyecto es obtener insights para un mejor entendimiento sobre cómo se compone el catálogo de Spotify. Para ello atravesé varios procesos como lo son la creación de la base de datos, la importación de los mismos al editor de código, la comprensión del contenido de cada una de las columnas, la limpieza de duplicados, la formulación de varias consultas a partir de un Lenguaje de Consultas Estructurado (SQL), la conexión entre las propias consultas y la herramienta de visualización de datos y por último la elección de los gráficos adecuados para hacer efectivo el *storytelling*. 

## 🗄️ Origen de los Datos
El dataset utilizado para este proyecto es de dominio público y fue extraído de Kaggle. Fuente original: [Spotify Tracks Dataset](https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset).
Cabe destacar que elegí un conjunto de datos ya limpio para poder enforcarme en el análisis exploratorio
